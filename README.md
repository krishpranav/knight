# knight
web application security scanning framework built with ruby 

[![forthebadge](https://forthebadge.com/images/badges/made-with-ruby.svg)](https://forthebadge.com)

# Installation
```
$ git clone https://github.com/krishpranav/knight
$ cd knight
$ make install
$ gem install bundler
$ bundle update
$ bundle install
$ ./knight --version
```
